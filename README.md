Python Port Scanner
Overview

This project is a Python-based port scanner that scans a target host for open ports. It demonstrates fundamental networking concepts and introduces basic techniques used in cybersecurity and network administration.

The scanner attempts to connect to specified ports and reports which ports are open and potentially accepting connections.

Features
Scans a target IP address or hostname
Detects open TCP ports
Displays scan results in a simple format
Demonstrates socket programming concepts
Lightweight and easy to use
Technologies Used
Python 3
Socket Programming
Networking Fundamentals
Purpose

This project was created to learn about computer networking, TCP/IP communication, and the basics of security assessment.

Example

Input:

Target: scanme.nmap.org
Ports: 20-100

Output:

Port 22: Open
Port 80: Open
